## On This Project: Chinese Chess

### Product Vision
Create an accessible yet deep Chinese Chess experience that serves both beginners learning the game and experienced players seeking a challenging AI opponent.

### Key User Segments
1. **Learners** — New to Chinese Chess, need tutorials and hints
2. **Casual Players** — Play occasionally, want polished experience
3. **Serious Players** — Seek strong AI opposition, analysis tools

### Must-Have Features (MVP)
- Complete rules implementation (all piece movements, check/checkmate detection)
- Playable AI opponent (multiple difficulty levels)
- Clean, intuitive board interface
- Basic sound feedback

### Should-Have Features
- Skin/theme customization
- Move history and undo/redo
- Game save/load
- AI move hints for learners

### Could-Have Features
- Online multiplayer
- Puzzle mode (tactical challenges)
- AI analysis mode (post-game review)
- Tutorial system

### Out of Scope (for now)
- Live online tournaments
- Social features (friends, chat)
- Mobile native apps

### Success Metrics
- **Engagement**: Average session length > 10 minutes
- **Retention**: 40% of users return within 7 days
- **Skill Progression**: Users improve win rate against AI over time
- **Accessibility**: New users complete first game without external help
